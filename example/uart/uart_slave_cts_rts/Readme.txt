功能描述
================================================================================
本样例主要展示串口从机硬件流控方式收发数据

说明：
1. 主机与从机连线方式如下：
    从机               主机
    PA02: Tx <-------> PA03: Rx 
    PA03: Rx <-------> PA02: Tx
    PC00:CTS <-------> PC01：RTS
    PC01:RTS <-------> PC00：CTS
         GND <-------> GND
2. 配置波特率9600bps
3. 与从机联调

使用步骤
1、用杜邦线将板子1的RX和TX分别与板子2的TX和RX连接，共电源共地
2、分别烧录主从机代码
3、在调试窗口观察数组u8RxData的值是否是0xaa,0x55，如果是表示运行正常
