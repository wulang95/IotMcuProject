功能描述
================================================================================
本样例主要展示ADT低功耗模式自动刹车。
说明：
    

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）示波器观测PA09(TIM5 CHA)和PB00(TIM5 CHB)端口输出波形，开始运行时端口输出正常PWM波形
3) 按user按键，进入低功耗模式后，LP刹车工作，端口输出预设低电平



================================================================================
注意
================================================================================
1) 等待User按键按下（PA07端口电平变低），进入深度休眠模式，
   为防止不能再次下载样例程序，该等待语句在例程中不可删除
