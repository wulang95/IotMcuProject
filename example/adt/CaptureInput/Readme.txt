功能描述
================================================================================
本样例主要展示ADT捕获输入功能
TIM4 CHA 输入PWM脉冲，捕获高电平和低电平的时间计数值
说明：
    

================================================================================
测试环境
================================================================================
测试用板: 
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）外部输入PWM连接PA08(TIM4 CHA)
2）打开工程编译并运行。
3）Live Watch（IAR）或在Watch（KEIL）观察u16CaptureA和u16CaptureB，
4）u16CaptureA捕获PWM脉冲的低电平时间计数值，u16CaptureB捕获PWM脉冲的高电平计数值
