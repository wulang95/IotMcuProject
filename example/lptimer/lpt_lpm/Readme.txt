功能描述
================================================================================
本样例主要展示低功耗Timer的计数功能。
说明：
本样例展示了LPTIMER0在深度睡眠模式下的计数功能，使用RCL时钟从0开始计数到0xffff溢出时产生中断，
MCU进入中断改变LED的状态(亮500ms->灭)，然后继续进入深度睡眠模式。
================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行；
4）程序会等待user key按下，在user key按下之后LPTIMER0开始运行并且MCU进入深度睡眠模式。
5）在深度睡眠模式下，LPTIMER0继续运行，计数溢出后产生中断，退出深度睡眠模式进入中断函数
改变LED的状态，清除中断标志位后退出中断并继续进入深度睡眠模式。
6) 信号定义： user key——PA07  LED——PD14  

================================================================================
注意
================================================================================
该Demo基于配套STK板将空置的GPIO均配置为输入，已最大限度降低功耗。
芯片进入深度休眠模式后，可将电流表串入Pwr Test的两个PIN脚测试MCU的功耗
