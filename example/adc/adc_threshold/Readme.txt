功能描述
================================================================================
本样例主要展示ADC单次一直采样区间值比较示例，使用中断方式获取ADC运行状态和ADC的采样值。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行
2）在ADC输入PIN PA00输入不同的电平。
3）调整ADC输入PIN(PA00)电平值进入不同的区间。
4）在ADC中断判断比较标志，如果在设定区间(u32AdcLowThd,u32AdcHighThd)内，读取转换值。
