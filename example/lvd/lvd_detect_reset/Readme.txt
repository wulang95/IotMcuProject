功能描述
================================================================================
本样例主要展示MCU的低电压检测（LVD）的复位功能

说明：
调整PB07输入电压，触发LVD低电压复位功能

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  连接PB07到直流电源，直流电源输出1.9v以上。
2）  打开工程编译并运行。
3）  调整直流电源到1.7V以下(因PB07为负端，故此时表现为高电平触发，MCU不停复位)。
4）  调整直流电源到1.9V以上。
4）  LED亮1s后熄灭，表示检测到上次复位为LVD复位。
5）  调整直流电源到1.9V以上并稳定，LED无变化。
