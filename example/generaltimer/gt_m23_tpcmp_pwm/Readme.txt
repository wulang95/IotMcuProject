功能描述
================================================================================
本样例展示Base Timer的CHA与CHB输出独立PWM的双点比较功能，
使用CCR0A和CCR0B控制OCREFA（CHA）的比较输出功能（双点比较功能，非对称互补PWM）,
CCR0B控制OCREFB（CHB）的比较输出功能

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
双通道示波器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）示波器探头连接到PA00(TIM0 CHA)和PA01(TIM0 CHB)
4）运行，观察波形输出
5）设置间隔计数为1，在UEV更新中断内对CCR0A和CCR0B的进行更改，CHA与CHB的占空比相隔
一个周期更替变化。
可以看到，CHA的上升沿时机由CCR0A设定的值控制，CHA的下降沿时机由CCR0B设定的控制。

================================================================================
注意
================================================================================
1）该样例展示的是CHA与CHB输出独立PWM的双点比较功能（CHA的左右边沿分别CCR0A和CCR0B设置）

2）如果要测试CHA与CHB的互补PWM功能（双点比较：CHA占空比由CCR0A和CCR0B共同设置，CHB与CHA互补），
请将stcBtBaseCfg.enPWMTypeSel  = BtIndependentPWM; ==》改为 stcBtBaseCfg.enPWMTypeSel  = BtComplementaryPWM;

3）如果要测试CHA与CHB的互补PWM功能（单点比较：CHA占空比由CCR0A设置，CHB与CHA互补）
请将stcBtBaseCfg.enPWMTypeSel  = BtIndependentPWM; ==》改为  stcBtBaseCfg.enPWMTypeSel  = BtComplementaryPWM;
stcBtBaseCfg.enPWM2sSel    = BtDoublePointCmp;     ==》改为  stcBtBaseCfg.enPWM2sSel    = BtSinglePointCmp;
