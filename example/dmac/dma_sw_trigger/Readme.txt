测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并下载程序。
2）在main函数while(1)处加断点。
3）运行程序。
4）如果程序能运行到while(1)断点处，则表示DMA功能运行正常。
