功能描述
================================================================================
本样例为CAN外部回环模式通信样例。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
PEAK_CAN-USB（或其它USB-CAN通信转换工具）

辅助软件:
---------------------
PCanView（或其它CAN协议分析工具（PC软件））

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3) 观察发送数据是否被正常接收（stcRxFrame.Data 是否等于 stcTxFrame.Data)。

================================================================================
注意
================================================================================
该模式下，CAN IP的TX和RX通过外部PHY短接，发送的数据直接通过RX被接收。
