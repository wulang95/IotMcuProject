功能描述
================================================================================
本样例为CAN查询收发数据样例。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
PEAK_CAN-USB（或其它USB-CAN通信转换工具）

辅助软件:
---------------------
PCanView（或其它CAN协议分析工具（PC软件））

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）连接硬件测试板与通信工具的CAN_H和CAN_L差分通信接口；
4）配置上位机为1M通信速率；
5）使用上位机发送数据到测试板；
6）测试板返回相同长度及内容的数据。
