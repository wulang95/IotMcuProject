功能描述
================================================================================
本样例主要展示Base Timer的定时功能。
说明：
本样例使用定时器定时，定时溢出后产生中断，并重新载入初值进行定时，并再次溢出进入中断。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
无

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）重载模式下，TIM0从ARR计数到溢出，会产生溢出中断；
4）在中断内对PD14输出进行翻转，控制LED等亮灭；
