功能描述
================================================================================
本样例主要展示GPIO的FAST IO输出功能。
说明：
FAST IO速度可以达到单周期翻转速度；

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
万用表

辅助软件:
---------------------

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）将程序下载至Demo板；
4）重新上电;
5）GPIO每个时钟周期输出翻转一次，可观测到端口PA01输出波形频率在HCLK=4MHz的情况下，
输出波形频率为2MHz；


================================================================================
注意
================================================================================
要最大限度实现GPIO的FAST翻转，对程序的写法要求比较高，请尽量参考样例配置和写法，
或者使用汇编连续的STR指令来实现。
