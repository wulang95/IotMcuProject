功能描述
================================================================================
本样例主要展示ADC 单次采样一直转换模式示例，可以随时获取当前的采样值。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并运行。
2）ADC将持续进行采样转换；
2）在ADC输入PIN(PA00)输入不同的电平；
3）读取转换结果。
4）在主循环里面读取(u32AdcRestult)转换结果。



================================================================================
注意
================================================================================
实际应用中，开启单次采样一直转换模式后，可根据应用需求随时读取当前采样结果。
