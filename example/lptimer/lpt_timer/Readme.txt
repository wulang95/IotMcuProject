功能描述
================================================================================
本样例主要展示低功耗Timer的计数功能。
说明：
本样例对展示低功耗Timer在低功耗模式下的带有门控功能的定时器功能，使用RCL时钟从0到
0xFFFF，溢出产生中断，退出低功耗模式并将LED控制信号翻转，中断处理结束后回到低功耗
模式，继续计数直到下次中断。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）将PB03接高电平
4）运行；
5）程序会等待user key按下，只有user key按下之后LPT运行并且进入超低功耗模式。
6）在超低功耗模式下，LPT继续运行，
7）当PB03接高电平时，LPT计数溢出后，在LPT中断里面翻转LED端口的输出电平,并继续进入超低功耗模式
8）当PB03接低电平时，LPT内部计数器停止计数。LED端口电平不会翻转。一直处于超低功耗模式。
9）信号定义： user key——PA07  LED——PD14  
================================================================================
注意
================================================================================
1.若芯片处于低功耗模式，则芯片无法使用SWD进行调式和下载功能。故如需要再次下载
调试其他程序，需要将芯片复位，并且复位后下载程序前不能按下user key。
