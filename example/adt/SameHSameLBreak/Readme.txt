功能描述
================================================================================
本样例主要展示ADT输出电平同高同低刹车功能。
说明：
    

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LF07x_EVB

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）示波器连接PA08(TIM4 CHA)与PA11(TIM4 CHB)端口，
2) 打开工程编译并运行
3) PA08和PA11输出同相的电平，
3）当同高检测使能，并且端口输出电平同高时，同高同低标志置起
4）当同低检测使能，并且端口输出电平同低时，同高同低标志置起

================================================================================
注意
================================================================================
1）示波器上可以查看到：

1.1) 同高测试时：

   PA08与PA07初始输出PWM,延迟2秒后，同高检测使能，
   检测到同高，停止输出；
   
   然后同高检测清除，TIM4 CNT清零，并进入下一测试

1.2) 同低测试时：

   PA08与PA07初始输出PWM,延迟2秒后，同低检测使能，
   检测到同低，停止输出
   然后同低检测清除，TIM4 CNT清零

2）当同高(同低)中断使能的时候，产生同高(同低)时，会立即进入中断，不论无效刹车条件1是否使能。
