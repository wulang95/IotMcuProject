功能描述
================================================================================
本样例主要展示低功耗Timer的计数功能。
说明：
本样例主要展示低功耗定时器LPTIMER在深度睡眠模式下对外部时钟进行计数，当溢出的时候翻转LED
的输出状态(亮500ms→灭)；在发生溢出中断后处理完中断继续进入深度睡眠模式。
================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32Lx7xGO-STK-V1.0

辅助工具:
---------------------
能够产生高低电平的信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）准备外部时钟源信号
4）运行；
5）程序会等待user key按下，在user key按下MCU会进入深度睡眠模式。
6）在超低功耗模式下，若PB04输入外部时钟信号，LPTimer会从65530进行计数，直到溢出产生中断
7) 信号定义： user key——PA07  LED——PD14  
================================================================================
注意
================================================================================
该Demo基于配套STK板将空置的GPIO均配置为输入，已最大限度降低功耗。
芯片进入深度休眠模式后，可将电流表串入Pwr Test的两个PIN脚测试MCU的功耗。
